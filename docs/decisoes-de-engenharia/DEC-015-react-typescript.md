## Frontend

```
React + Vite + TypeScript
```

Motivos:

- stack moderna;
- rápida;
- boa organização;
- muito usada no mercado;
- prepara você para projetos profissionais.

---

## Estilização

```
Tailwind CSS
```

Motivos:

- produtivo;
- ótimo para criar dashboards;
- reduz CSS solto;
- facilita responsividade.

---

## DevOps

```
Docker + Docker Compose
```

Motivos:

- padroniza ambiente;
- facilita deploy;
- integra backend, frontend e banco.
