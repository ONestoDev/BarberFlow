## DEC-010 — Fluxos antes de Telas

Uma prática comum é começar desenhando telas.

Nós faremos o contrário.

Primeiro validaremos os fluxos de negócio.

Depois construiremos:

- Banco de Dados
- Backend
- Frontend

Essa ordem reduz retrabalho e garante que a interface reflita o comportamento esperado do sistema.
