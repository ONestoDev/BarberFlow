# Uma decisão de produto

Aqui quero tomar uma decisão que considero muito importante.

## O Cliente NÃO terá acesso ao sistema administrativo.

Ele terá uma área própria.

Isso significa que futuramente teremos dois Frontends.

```
Frontend Administrativo

↓

Administrador

↓

Barbeiro
```

e

```
Portal do Cliente
```

Ou até mesmo:

```
Aplicativo Mobile
```

Mas isso será uma evolução futura.

Na V1, manteremos tudo num único frontend, com acesso controlado por perfis, para reduzir complexidade.
