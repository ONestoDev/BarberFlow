## DEC-002

### Utilizaremos RBAC (Role-Based Access Control)

Papéis:

```
ADMIN

BARBEIRO

CLIENTE
```

Cada usuário possuirá exatamente um papel na primeira versão.

### Por quê?

Porque:

- é simples;
- é escalável;
- atende nosso MVP;
- pode evoluir facilmente para múltiplos papéis por usuário no futuro.
