## DEC-006 — Simplicidade como princípio arquitetural

Toda nova tecnologia deverá responder a três perguntas:

1. Qual problema ela resolve?
2. Ela reduz ou aumenta a complexidade?
3. Ela faz sentido para o estágio atual do produto?

Se a resposta não for convincente, ela ficará para uma versão futura.

Essa decisão será um dos pilares do projeto.
